v0.1.0 Built with Tauri + React

A better terminal
for AI development

Smart tabs, split panes, a thoughts scratchpad, and 8 built-in themes. Designed for developers who work with agentic AI in the CLI.

Download for macOS View on GitHub
$ git clone https://github.com/alvinreyes/better-terminal && cd better-terminal && npm i && npm run tauri build

Everything you need

⌨

Smart Tabs

Named tabs with instant switching via Cmd+1-9. Double-click or Cmd+R to rename. Navigate with Cmd+Shift+[/].

▦

Split Panes

Split horizontally or vertically. Terminals persist across splits without resetting. Drag dividers to resize.

✎

Thoughts Scratchpad

Type your thoughts and send directly to the terminal with Cmd+Enter. Save as reusable notes with Cmd+S.

⚙

8 Themes

GitHub Dark, Dracula, Monokai, Nord, Catppuccin, Solarized, Tokyo Night, One Dark. Plus custom color overrides.

Aa

Font Customization

Choose from 7 monospace fonts. Adjust size from 10-24px. Tune line height and cursor style.

💾

Save Workspaces

Save your current tab layout as a named workspace. Restore it later with one click.

Keyboard-first

Every action has a shortcut. No mouse required.
